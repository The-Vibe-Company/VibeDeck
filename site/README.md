# Site vitrine VibeDeck

Landing page statique du projet (design « Site VibeDeck v2 », Claude Design). Aucun build : un seul `index.html`.

- Prévisualiser : `open index.html` (ou `python3 -m http.server` dans ce dossier).
- Déployer : `vercel deploy --prod` depuis ce dossier (projet Vercel `vibedeck-site`).
